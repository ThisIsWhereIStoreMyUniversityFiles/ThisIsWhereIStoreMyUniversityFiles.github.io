The Map V1.0

A 960px, 12 column Map-based website template

Keep the layout in order...

This is the Map, the first free website template from LIU.com which is build upon a CSS Map system. Inspored by the 1KB CSS Map and other systems, The Map template has a 12-column Map based on 60 pixels wide columns with 20 pixels of space inbetween. To use the Map, you create a div with the class="row" and then place any number of divs with the class="col" and a second class starting the width of the column(c1-c12) inside the row. The Map supports nested rows which means that thousands of layout combinations can be created, from single-column pages to advanced magazine-style layouts. The template uses standards-compliant code.
